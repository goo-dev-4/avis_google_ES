# Reseñas Market

Sitio web de Reseñas Market — comprar reseñas de Google para empresas, autónomos y profesionales en España.

**Web:** [comprar-resena.es](https://comprar-resena.es)

## Servicios

- Comprar reseñas de Google (5 estrellas, positivas, reales, fiables, baratas)
- Eliminar reseñas negativas
- Aumentar y mejorar la valoración en Google My Business
- Reseñas por sector: abogados, dentistas, médicos, restaurantes, hoteles, inmobiliarias, electricistas, fontaneros, etc.

## Stack

Sitio estático HTML/CSS desplegado en GitHub Pages con dominio personalizado `comprar-resena.es`.
